
# TheAria — Portfolio

> A modern, full-screen portfolio for a Frontend Developer & UI/UX Designer, built with vanilla HTML, CSS, and JavaScript.
> Features a vertical-scroll section engine, custom cursor, glassmorphism UI, and a typewriter code card.

![Portfolio Preview](assets/preview.png)

---

## 🚀 Live Demo

 https://thewestboy.github.io/thearia-portfolio/ 

---

## ✨ Features

- Full-screen section navigation with smooth vertical scrolling
- Custom cursor with interactive hover effects
- Glassmorphism UI and animated gradient backgrounds
- Typewriter-style animated code card
- Dynamic role rotator
- Animated skill bars
- Draggable portfolio/project carousel
- Contact form with front-end validation
- Live local time and availability status
- Fully responsive layout
- Reduced motion support for accessibility

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|

| CSS3 (Custom Properties, Grid, Flexbox) | Styling, layout, animations |

## 🧩 Getting Started

### Clone the repository

```bash
git clone https://github.com/thewestboy/thearia-portfolio.git
cd thearia-portfolio
```

### Run locally

Open `index.html` directly in your browser.

Or use a local development server:

```bash
npx serve .
```

Then visit:

```text
http://localhost:3000
```
